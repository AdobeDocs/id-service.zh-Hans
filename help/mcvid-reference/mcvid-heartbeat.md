# 视频心率和 Experience Cloud ID 服务 {#video-heartbeat-and-the-experience-cloud-id-service}

要使用视频心率，必须安装 Experience Cloud ID 服务。测量 JavaScript 或 Flash 上的视频时，如果在网站页面上以及视频播放中实施 ID 服务，则可以确保访客在查看视频时不会被另行计数。
